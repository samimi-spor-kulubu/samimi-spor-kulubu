## Teknoloji Stack
- **Framework:** Next.js (App Router) + TypeScript
- **Stil:** Tailwind CSS v4
- **UI Kit:** shadcn/ui (Radix / Nova)
- **i18n:** next-intl — TR varsayılan, EN `/en/` altında
- **Veritabanı / Auth:** Supabase
- **Deploy:** Vercel

> ⚠️ Bu Next.js sürümünde breaking change'ler var. Kod yazmadan önce `node_modules/next/dist/docs/` içindeki ilgili rehberi oku (bkz. AGENTS.md).
